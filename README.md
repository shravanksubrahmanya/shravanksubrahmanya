## About This Portfolio

This portfolio is built with a modern tech stack and features an interactive AI robot guide that flies across the screen, responds to card interactions, and introduces each section.

**Tech Stack:** React 19 + Vite 6 + Tailwind CSS 4

**Features:**
- Interactive floating robot guide with Disney-style squash-and-stretch animations
- Section-aware waypoint system with trail particles
- Card-click robot interactions across all sections
- Light / Dark theme toggle with CSS custom properties
- Aurora background orbs and glassmorphism design
- Animated stat counters and typing effects
- Fully responsive and mobile-friendly
